# PDF Files Directory

This directory contains PDF textbooks and study materials organized by class.

## Directory Structure

```
public/
  pdfs/
    class6/
      Class6_Mathematics_1_Knowing_Our_Numbers.pdf
    class7/
    class8/
    ...
```

## Adding PDF Files

1. Place your PDF file in the appropriate class directory (e.g., `class6/`)
2. Name the file exactly as specified in the syllabus data:
   - For Class 6 Maths Chapter 1: `Class6_Mathematics_1_Knowing_Our_Numbers.pdf`
3. The PDF will automatically appear in the subject detail page

## Supported File Names

The system will try to find PDFs using these patterns:
- Exact filename as specified in syllabus
- Lowercase with spaces replaced by hyphens
- Common alternative names

## File Size Recommendations

- Keep PDF files under 50MB for optimal loading
- Use compressed PDFs when possible
- Consider splitting large textbooks into chapters
